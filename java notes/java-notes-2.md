# Privacidade de código

Precisamos proteger as variáveis dos objetos depois de sua criação, e não somente na sua criação.

# Organização

## Organização por convenção de classes

```
  class MyClass {
    |> atributos
    |> construtores
    |> getters and setters
    |> métodos
    |> toString 
  } 
```
