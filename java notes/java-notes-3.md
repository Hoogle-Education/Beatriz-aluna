# Manipulando muitos dados

## Vetor

é um conjunto de valores agrupados por um nome.

vetor - n posições

1. O vetor é acessado da posição `0` até a `n-1`.

2. O vetor é homogeneo.

declaração:

```java
  int[] name = new int[size];
```

## Switch

swith é um if-else melhorado.

sintaxe:
```java
  switch (value){
    case 1:
      // your code
      break;
    case 4:
      // your code
      break;
    default:
      //
      break;
  }
```
